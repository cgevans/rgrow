# Models

rgrow supports several tile assembly and related models:

- [kTAM](ktam.md) — Kinetic Tile Assembly Model
- [aTAM](atam.md) — Abstract Tile Assembly Model
- [Old kTAM](oldktam.md) — Legacy kTAM implementation
- [SDC](sdc.md) — 1D Strand Displacement Circuits
- [KBlock](kblock.md) — Block-based model
