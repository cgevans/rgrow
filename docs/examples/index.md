# Examples

Interactive Jupyter notebook examples demonstrating rgrow's capabilities.

## Tile Assembly

- [Sierpinski Triangle](simple-sierpinski.ipynb) — Classic Sierpinski triangle assembly using the kTAM
- [Tube Canvas](tube.ipynb) — Assembly on a tube (periodic) canvas

## Forward Flux Sampling

- [FFS - Simple](ffs-simple.ipynb) — Basic forward flux sampling for nucleation rate calculation
- [FFS - Pattern Recognition](ffs-pattern-recognition.ipynb) — FFS applied to pattern recognition tile sets (Evans 2024)

## Specialized Models

- [KBlock](kblock.ipynb) — Block-based tile assembly model
- [SDC - Bit Copy](sdc-bit-copy.ipynb) — 1D strand displacement circuit: bit copying
- [SDC - Bit Copy Dual](sdc-bit-copy-dual.ipynb) — 1D strand displacement circuit: dual bit copying
